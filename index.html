01 / HELLO, PROPERLY

我不是一个
淡人。

我喜欢人群里突然冒出来的笑声，喜欢一群人把一件事做成，也喜欢一个人坐在咖啡馆，把没想明白的事慢慢想清楚。

我会焦虑，会迷茫，广告也会跑差，AI 也会卡在做不出来的那一步。但我大多时候还是会继续：想到什么就去做，不会就去学，失败了就再试一次。

我相信事在人为。比起宿命，我更相信选择，也努力把自己的选择变成一个好答案。

02 / LIFE JOURNAL

没有大事，
也值得记一下。

生活不是一条需要汇报的时间线。
它更像一些忽然想留下来的瞬间。

AN ORDINARY DAY

今天什么都没发生。

但也没有关系。日子有时候就是该留一点空白。

PAUSETONIO'S NOTE

但都是我真的会记住的事。

04 / A FEW THINGS I BELIEVE

我欣赏稳定、
责任和韧性。

我喜欢乌尔苏拉。不是因为她有多完美，而是因为她能撑住一个家。那些不动声色的承担、把事情放在心上的认真，我很敬佩。

但我不是乌尔苏拉。我骨子里还是喜欢探索、创造、折腾和新的东西。所以 AI、产品、创业，都会让我忍不住想靠近一点、试一试。

成长不是变成另一个人是更会和自己一起生活

05 / STILL LEARNING

没有成熟叙事，
只有正在发生。

以前遇到问题，
会把自己关起来。

现在也会焦虑，只是会去解决。以前总想先得到答案，现在慢慢接受：很多答案不是想出来的，是走出来的。

生活还是状况百出。感情偶尔会牵动情绪，工作也会出错。但我更愿意让它们发生，然后继续生活。

今天不知道会发生什么，
但最好发生一点有意思的事情。

END / FOR NOW

我的人生不是康庄大道，
也不是独木桥。

我不知道什么是最好的生活。
但我知道，
我一直在路上。
